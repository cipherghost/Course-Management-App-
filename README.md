# COURSE MANAGEMENT APP 

## Technologies used
* HTML
* CSS
* JS
* PHP
* Apache
* MySQL

## Database table design

Database consists of 4 tables with following structure :

```
+---------------+--------------+------+-----+-------------+----------------+
| Field         | Type         | Null | Key | Default     | Extra          |
+---------------+--------------+------+-----+-------------+----------------+
| id            | int(11)      | NO   | PRI | NULL        | auto_increment |
| dname         | varchar(150) | YES  |     | NULL        |                |
| date          | date         | YES  |     | NULL        |                |
| nod           | int(11)      | YES  |     | 0           |                |
+---------------+--------------+------+-----+-------------+----------------+
```
